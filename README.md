# Конвертер выгрузки из ФГИС "АРШИН" в ФГИС ФСА
Небольшой и простой веб скрипт который позволит вам практически автоматизировать конвертацию выгрузки из ФГИС "АРШИН" для поледующей загрузки в ФГИС ФСА.

Особенности:
- Не требует уствновки.
- Не передаёт данные, всё выполняется у вас локально на устройстве.
- Ведет базу данных по сопоставлению МПИ и СНИЛС сотрудников, ввести надо только один раз.
- Проверяет сопоставления и выводит запросы только на недостающие данные.
- Может выгружать ранее введенные данные сопоставлений в формате JSON для переноса на другое устройство.
- Детальный лог и отчёты конвертации.
- Проверка валидности данных.
- Поддержка пакетной загрузки сопоставлений МПИ из таблиц.
- Абсолютно бесплатна.

Использование: https://fgis.freize.net/
